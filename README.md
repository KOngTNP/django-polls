## Requirements
 * Python 3.6 or newer
 * Django 2.1.2 or newer
 * Python add-on modules as in [requirements.txt](requirements.txt)
 
 
## How to Run
 1. git clone https://github.com/KOngTNP/django-polls.git
 
 2. cd django-polls
 
 To run server
 3. python3 manage.py runserver
 
 To tests
 4. python3 manage.py test polls
